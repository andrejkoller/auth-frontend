## Tech stack
- Next.js (app router)
- React + TypeScript
- MUI (Material UI)
- Axios for API calls

## Prerequisites
- Node 18+
- npm / pnpm / yarn

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/andrejkoller/auth-frontend.git
cd auth-frontend
```
### 2. Install dependencies

```bash
npm install
```
### 3. Start development server

```bash
npm run dev
```
